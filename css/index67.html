flex – A CSS layout model that arranges items in a row or column. justify-content – Defines how space is distributed along the main axis (like center, start, end). flex-wrap – Allows flex items to wrap onto multiple lines when needed. flex-basis – Sets the initial size of a flex item before remaining space is distributed. container – A wrapper element that holds all the .item divs and controls their layout.
